## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/eyenine/quran-web-app.git
   cd quran-web-app
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Start the development server:**
   ```sh
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
